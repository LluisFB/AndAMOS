# AndAMOS
This is the GIT repository for the AndAMOS (https://www.cima.fcen.uba.ar/proyectos/AndAMOS/) iniciative from the GEWEX GHP ANDEX (https://www.gewex.org/project/andex/) program (full Spanish version https://www.andex-rhp.org/).

AndAMOS aims to analyze the current modelling chains in providing useful data in direct demand from impact and stakeholders. In order to achieve the goal a series of 'Experiencies' are designed. These experiences target specific demands of impact and/or stakeholders and encompass all the necessary modeling steps needed to provide actionable data.

Currently there are 2 experiencies:
- Tropical North: Severe drought in Ecuador and landslides in Colombian Andes
- Malargüe: Zonda wind (Argentinean name for strong Foehn wind) characterization and extreme wind exposure in Malargüe
